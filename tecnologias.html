PHP - Laravel

De longe a linguagem de programação que mais utilizei para fazer projetos até o momento. Desenvolvi o Back-End completo do Projeto "Site Faculdade" no PHP puro. Assim como o projeto "ProjetoO", que visava uma recriação da OLX.

Em Laravel, desenvolvi para uma entrevista técnica, um site de "Votações" onde tinham algumas enquetes e os usuários podiam participar.

Java

Minha primeira linguagem de programação, onde aprendi a toda lógica de programação. Tenho projeto "Algoritmo de busca" onde fiz um código de Busca em Largura, com o intuito de encontrar um determinado arquivo/aplicativo que o usuário fornecer o nome.

Tecnologia de armazenamento de dados, onde utilizei em todos os meus projetos que contém algum tipo de banco de dados e relação de dados. Majoritariamente utilizado no terminal.
