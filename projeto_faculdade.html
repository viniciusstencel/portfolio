Site Informativo para Faculdade

Site Dinâmico para as comodidades e eventos da Faculdade.

Esse projeto foi inteiramente feito em PHP puro, site completamente dinâmico com banco de dados.

Desenvolvi todo o Back-End do site; Login, Cadastro de usuários, Cadastro de eventos e Atléticas, o site contém um sistema de nenhuma pessoa pode ter o cadastro "ativo" se o administrador do site não liberar, a mesma coisa se aplica para o cadastro de atléticas e eventos, a parte de comodidades pode ser cadastrada apenas pelo administrador.
